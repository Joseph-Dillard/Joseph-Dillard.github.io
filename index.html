Quick Application: Remediate Webpage

Overview

This webpage has many accessibility errors. You have been exposed to all of the issues found here.

Allowed Resources

  1. You can use any course and online resources as long as you work alone.
  2. Accessibility trumps other aspects of the presentation!
    The accessible version you create likely won't be exactly the same as the original. Regardless, it should be structurally and semantically equivalent.
  3. Hint: The activities we did in class and material from the textbooks is sufficient reference material. To find this information on Canvas, Canvas.

What to do

  1. Download a copy of the page to edit its HTML.
    1. You can generally do this by viewing the page in a broswer and choosing to "save page as" e.g. use ctrl+s.
  2. Resolve the accessibility issues.
    1. There are seven categories of accessibility issues. Some categories have more than one instance of a problem.
    2. Rubric:
      1. You start with 3 points for attempting to resolve the issues regardless of success.
      2. For each issue:
        1. You get one point for completely resolving the issue.
        2. You get half a point for partially resolving the issue. For example fixing it in one place but not in other places. Not all issues can be partially resolved.
      3. Total of 10 points.
  3. Submit the updated HTML file.
    1. You may also include external CSS files with your submission if you do the challenge, but feel free to use internal CSS. Web page with reminders of ways to add CSS: w3schools tutorial
  4. As a challenge, use CSS to make the accessible html render more like the original.
